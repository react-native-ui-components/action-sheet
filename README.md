# @react-native-ui-components/action-sheet

Action sheet component for React Native

## Installation

```sh
npm install @react-native-ui-components/action-sheet
```

## Usage

```js
import { ActionSheet } from '@react-native-ui-components/action-sheet';

// ...

ActionSheet.open({
  actions: [{ title: 'Hi' }],
});
```
